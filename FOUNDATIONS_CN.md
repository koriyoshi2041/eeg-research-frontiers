## 为第二部分（EEG 基础概念）的基础补充

### B1.1 什么是"振荡"？——从弹簧到大脑

**为什么要学这个**：EEG 研究的核心对象就是"神经振荡"——大脑产生的有节律的电信号。不理解振荡，就看不懂 EEG 的任何分析方法。

**从物理类比开始**。

一根弹簧被拉伸后松手，会来回弹动。这个"来回"就是振荡。它有三个基本属性：

- **频率（Frequency）**：每秒弹几次。单位：赫兹（Hz）。弹簧硬 → 频率高；弹簧软 → 频率低。
- **振幅（Amplitude）**：每次弹多远。拉得越开，振幅越大。
- **相位（Phase）**：在"来回"周期中的当前位置。两根弹簧可以同频率同振幅，但一个在最高点时另一个在最低点——这就是"相位差"。

**大脑的振荡**来自一个更精妙的机制：兴奋性神经元和抑制性神经元之间的相互作用。

想象一群人在做"人浪"：
1. 兴奋性神经元（E）先激活 → 放电
2. E 的放电同时激活了抑制性神经元（I）
3. I 反过来抑制了 E → E 停止放电
4. I 的抑制效果逐渐消退 → E 恢复激活
5. 回到第 1 步

这个 E→I→E→I 的循环自然产生了振荡。循环快（I 恢复快）→ 高频（gamma，30-100 Hz）。循环慢（I 恢复慢，或涉及远程连接如丘脑）→ 低频（alpha/delta）。

**关键直觉**：大脑的振荡不是"有人在驱动它振动"，而是**兴奋和抑制之间的自然平衡**产生的涌现现象——就像弹簧不需要外力就会自己弹，只需要被"拉开"（激活）一次。

**参考**：Buzsáki, *Rhythms of the Brain* (Oxford, 2006)，第 4-6 章。

### B1.2 频域分析入门——为什么要做傅里叶变换

**为什么要学这个**：EEG 的五个频段（delta/theta/alpha/beta/gamma）是通过频域分析定义的。不理解傅里叶变换，就不知道这些频段是怎么分出来的。

**核心思想**：任何信号都可以分解为不同频率的正弦波之和。

类比：白光通过棱镜分解为彩虹——每种颜色是一个频率。傅里叶变换就是信号的"棱镜"。

```
时域信号 x(t) ──傅里叶变换──> 频域表示 X(f)

x(t) = 一团看起来杂乱无章的波形
X(f) = 每个频率上有多少"能量"（功率谱密度, PSD）
```

**EEG 中的应用**：

1. **功率谱密度（PSD）**：最基础的分析。画出"各频率的能量"，可以直接看到 alpha 峰（~10 Hz 处的突起）、beta 活动等。
2. **短时傅里叶变换（STFT）/ 小波变换**：加上时间维度——哪个频段在什么时候最活跃。输出是时频图（spectrogram），横轴时间，纵轴频率，颜色代表能量。
3. **1/f 谱**：EEG 的功率谱不是平的——低频能量高，高频能量低，大致按 1/f^β 衰减。这个斜率 β（谱斜率/spectral exponent）本身就是一个有意义的指标——它反映了兴奋/抑制平衡。

**为什么不能只看时域**：

原始 EEG 看起来像噪声——什么都看不出来。但做了频域分解后，你能看到：
- 闭眼时 alpha 功率飙升（"alpha 阻断"现象的反面）
- 深度睡眠时 delta 占主导
- 专注思考时 beta/gamma 增加

**关键直觉**：频域分析是 EEG 的"显微镜"——把混在一起的信号按频率分开，每个频率对应不同的脑状态。

### B1.3 电极放置的物理直觉——为什么位置很重要

**为什么要学这个**：EEG 的电极不是随便放的。放在不同位置，能"看到"大脑不同区域的活动。

**电场的基本性质**：

大脑产生的电场通过导电组织（脑脊液、颅骨、头皮）传播到体表。就像把一个小灯泡放在一碗半透明果冻里——从外面能看到光，但光会被散射、衰减。

- **正上方最亮**：电极离源最近，信号最强。
- **越远越暗**：信号随距离衰减（大致按 1/r² 到 1/r³ 衰减）。
- **颅骨是最大的"模糊器"**：颅骨的电导率只有脑脊液的 1/80，像一层毛玻璃——空间分辨率因此被限制在 ~1-2 cm。

**10-20 国际标准系统**：

1958 年由 Jasper 提出。"10-20"指的是电极间距占头围总长的 10% 或 20%。

命名规则：
- **字母 = 脑区**：F (Frontal, 额叶), C (Central, 中央), P (Parietal, 顶叶), O (Occipital, 枕叶), T (Temporal, 颞叶)
- **数字 = 半球**：奇数=左, 偶数=右, z=中线
- 例：`Cz` = 中央中线（运动皮层正上方），`O1` = 左枕叶（视觉皮层）

**为什么运动想象 BCI 看 C3/C4**：C3 在左运动皮层上方，C4 在右运动皮层上方。想象右手动 → 左运动皮层激活 → C3 处 mu 节律（8-12 Hz）衰减。这个衰减就是 BCI 解码的信号。

**关键直觉**：电极位置 = 你在"偷听"大脑的哪个区域。10-20 系统是一种标准化的"偷听位"布局。

### B1.4 ERP 的统计学基础——为什么要做平均

**为什么要学这个**：事件相关电位（ERP）是 EEG 认知研究的基石，但它的提取方法本质上是一个统计操作。

**问题**：你想知道"看到人脸时大脑有什么反应"。但单次试验的 EEG 里：
- 信号（大脑对人脸的响应）：~2-5 µV
- 噪声（其他脑活动 + 肌电 + 电源干扰）：~20-50 µV

信噪比（SNR）约为 1:10。单次试验根本看不到信号。

**解决方案：叠加平均（trial averaging）**。

给被试看 200 次人脸。每次记录 EEG。把 200 次信号对齐到"人脸出现"那个时刻，然后取平均。

为什么有效？因为：
- **信号是时间锁定的**：每次看到人脸，大脑在 ~170ms 后都会产生一个负波（N170）。这个信号在 200 次试验中重复出现在同一位置。
- **噪声是随机的**：其他脑活动不会恰好在每次人脸出现后 170ms 产生同样的波形。

```
平均后的信噪比 = 单次 SNR × √N

200 次平均 → SNR 提高 √200 ≈ 14 倍
原来 1:10 → 现在 14:10 ≈ 1.4:1 → 信号浮出水面
```

**注意**：这意味着 ERP 研究必须有**大量重复试验**（通常 100-200 次/条件）。单次试验的 ERP 检测是一个活跃的研究方向（deep learning single-trial ERP detection），但远未解决。

**关键直觉**：ERP 不是"看一次就能看到的东西"——它是从噪声中通过统计方法提取出来的。这既是 ERP 的力量（可以测量毫秒级的认知过程），也是它的限制（需要大量重复，不适合研究不可重复的认知事件）。

## 为第三部分（经典信号处理）的基础补充

### B2.1 滤波器基础——高通、低通、带通是什么

**为什么要学这个**：EEG 预处理的第一步就是滤波。不理解滤波器，无法理解数据是怎么被处理的。

**直觉**：滤波器就是一个"频率选择器"——让某些频率通过，挡住其他频率。

| 类型 | 做什么 | 类比 | EEG 中的用途 |
|------|--------|------|-------------|
| **低通滤波器** | 只让低频通过 | 磨砂玻璃（去掉细节/高频） | 去除高频肌电伪迹（>40 Hz） |
| **高通滤波器** | 只让高频通过 | 排水沟的栅格（去掉大块/低频） | 去除缓慢漂移（<0.1-1 Hz） |
| **带通滤波器** | 只让某个范围的频率通过 | 收音机调台 | 提取特定频段（如 alpha 8-13 Hz） |
| **带阻/陷波滤波器** | 挡住某个窄频段 | 消音器 | 去除电源线干扰（50/60 Hz） |

**重要注意**：滤波不是"免费"的。

- **因果滤波器**会引入相位延迟（信号变形）
- **零相位滤波器**（前向+后向）不引入延迟，但需要全段数据（不能实时用）
- **滤波阶数**过高会引入振铃效应（ringing）——假的振荡出现在滤波后的信号里
- ERP 研究中，高通滤波截止频率的选择（0.1 Hz vs 0.5 Hz vs 1 Hz）会显著影响结果——这是一个被反复讨论的方法学争议

**关键直觉**：滤波是最基本的信号处理工具，但也是最容易引入伪影的步骤。"怎么滤"本身就是一个需要谨慎决策的研究选择。

**参考**：Widmann et al., "Digital filter design for electrophysiological data" (*Psychophysiology*, 2015)。

### B2.2 独立成分分析（ICA）——盲源分离的直觉

**为什么要学这个**：ICA 是 EEG 预处理中去除伪迹（眼动、肌电、心跳）的核心方法，也是理解脑源分离的基础。

**经典类比：鸡尾酒会问题**。

房间里有 3 个人同时说话。你有 3 个麦克风放在不同位置。每个麦克风录到的是 3 个声音的混合。你能只从录音中分离出每个人的声音吗？

```
录音 1 = 0.8×声音A + 0.3×声音B + 0.1×声音C
录音 2 = 0.2×声音A + 0.7×声音B + 0.4×声音C
录音 3 = 0.4×声音A + 0.1×声音B + 0.9×声音C
```

ICA 的答案：**可以**，只要源信号之间是统计独立的。

ICA 假设：
1. **源信号相互独立**（你说你的，我说我的）
2. **混合是线性的**（每个麦克风录到的是各源信号的加权和）
3. **源信号不是高斯分布的**（高斯的独立信号混合后仍然高斯，无法区分——这是一个数学限制）

**在 EEG 中的应用**：

- "麦克风" = 头皮电极（20-256 个）
- "说话的人" = 脑源 + 伪迹源
- ICA 分离出独立成分（IC），每个 IC 是一个"源"
- 人工或自动（ICLabel）标记哪些 IC 是伪迹（眼动、肌电、心跳），去除它们
- 重建 = 去掉伪迹 IC 后把剩下的 IC 混合回去

**ICA 的局限**：
- 需要足够多的数据（经验法则：至少 20×通道数² 个时间点）
- 假设线性混合——颅骨非线性效应被忽略
- 成分数 = 通道数（不能分离超过通道数的源）
- 不同 ICA 算法（Infomax, FastICA, AMICA）可能给出不同结果

**关键直觉**：ICA 是 EEG 研究者最重要的工具之一——它把"一团混合的东西"分离成"各自独立的成分"。但它做的是**统计分离**，不是物理分离——分出来的成分不一定对应一个具体的脑区。

### B2.3 逆问题为什么"病态"——一个工程直觉

**为什么要学这个**：EEG 源定位（从头皮电压推断脑内源位置）是一个"逆问题"。理解它为什么难，才能理解为什么 EEG 的空间分辨率远不如 fMRI。

**正向问题（Forward Problem）**：

给定脑内的电流源位置和强度，计算头皮上每个电极看到的电压。

这是一个**有唯一解**的物理问题。只需要知道头部的几何形状和各层组织的电导率（脑、脑脊液、颅骨、头皮），就能精确计算。类比：知道灯泡的位置和亮度，计算墙上每个点的照度。

**逆问题（Inverse Problem）**：

给定头皮上所有电极的电压，推断脑内的源位置和强度。

这是一个**没有唯一解**的问题。Helmholtz 在 1853 年就证明了：无限多种不同的源分布可以产生完全相同的头皮电压分布。

工程类比：你在一个封闭房间外面，只能通过墙上 64 个小洞看到室内的光照强度。你能确定房间里有几盏灯、在哪、多亮吗？**不能**——一盏很亮的灯和两盏较暗的灯可以在你的 64 个观测点产生完全一样的光照分布。

**怎么办？加约束。**

不同的源定位方法 = 不同的"先验假设"：
- **最小范数估计（MNE）**：假设源的总能量最小 → 倾向于给出弥散的解
- **sLORETA / eLORETA**：假设相邻源强度相近 → 倾向于给出平滑的解
- **Beamforming**：假设不同脑区的信号不相关 → 空间滤波器
- **偶极子拟合**：假设只有 1-3 个集中的源 → 最受限但最精确（如果假设对的话）

**关键直觉**：EEG 源定位不是"测量"——而是在无限多个可能的答案中，用先验假设选一个"最合理"的。不同假设给不同答案。这是 EEG 空间分辨率有限的**根本原因**，不是工程限制。

### B2.4 图论与脑网络——为什么大脑是一张图

**为什么要学这个**：现代脑科学越来越关注"脑区之间的连接"而不只是"单个脑区的活动"。EEG 的连接性分析需要图论基础。

**大脑作为网络**：

大脑不是 86 亿个独立的神经元——它们通过白质纤维束连接成一个**复杂网络**。

| 概念 | 图论术语 | 大脑对应物 |
|------|---------|-----------|
| 顶点（Node） | 节点 | 脑区（或电极） |
| 边（Edge） | 连接 | 两个脑区之间的功能连接强度 |
| 度（Degree） | 一个节点连了多少条边 | 一个脑区和多少其他脑区有功能连接 |
| 聚类系数 | 你的朋友之间也是朋友的比例 | 局部脑区之间的紧密程度 |
| 最短路径 | 两个节点之间最少经过几跳 | 信息从一个脑区传到另一个需要多少步 |
| Hub（枢纽） | 度数特别高的节点 | 连接特别多的脑区（如后扣带皮层、楔前叶） |

**小世界网络（Small-World Network）**：

大脑网络既不是完全随机的，也不是完全规则的——它是"小世界"的：
- 高聚类系数（局部连接紧密，像小镇上人人认识）
- 短平均路径长度（全局信息传递高效，像航空网络）

这种结构在**高效性**（短路径→快速全局通信）和**韧性**（高聚类→局部损伤不致命）之间取得平衡。EEG 脑连接研究的一个经典发现：阿尔茨海默病患者的脑网络变得更"随机"——失去了小世界特性。

**EEG 怎么构建脑网络**：
1. 选择节点：电极位置（或通过源定位得到的脑区）
2. 计算边：对每对节点计算连接强度（相干性、PLV、Granger 因果等）
3. 阈值化：只保留强连接（避免噪声连接）
4. 计算图论指标：度、聚类系数、路径长度、模块化等

**关键直觉**：大脑不是一堆独立的"功能区"——它是一个动态变化的网络。EEG 的时间分辨率让它特别适合研究这个网络如何在毫秒尺度上重组。

## 为第四部分（神经科学前沿）的基础补充

### B3.1 意识的科学定义——哲学问题如何变成实验问题

**为什么要学这个**：EEG 用于意识研究是一个前沿方向，但"意识"这个词在日常生活和科学中含义非常不同。

**科学中的"意识"不是哲学辩论——它有操作性定义。**

| 术语 | 含义 | 如何测量 |
|------|------|---------|
| **清醒（Wakefulness）** | 眼睛睁着、能活动 | 行为观察 |
| **觉知（Awareness）** | 有主观体验（看到、听到、感受到） | 主观报告 或 脑信号推断 |
| **最小意识状态** | 间歇性觉知（偶尔响应指令） | 临床评估 + EEG |
| **植物状态/无反应觉醒** | 清醒但无觉知（睁眼但不响应） | 临床评估 + EEG/fMRI |

**EEG 测量意识的核心方法**：

- **PCI（扰动复杂性指数）**：用 TMS 刺激大脑，记录 EEG 的反应。有意识的大脑反应复杂且分化（PCI 高）；无意识的大脑反应简单或无反应（PCI 低）。准确率 97%+。
- **频谱斜率（1/f slope）**：有意识时 EEG 功率谱的斜率较平（更多高频活动）；无意识时斜率更陡（低频主导）。

**为什么这很重要**：每年有大量"植物状态"患者可能实际上有意识但无法表达。EEG 提供了一种**不需要患者主动响应**就能评估意识水平的方法。

---

### B3.2 睡眠的阶段与功能——为什么我们要睡觉

**为什么要学这个**：睡眠分期是 EEG 临床应用中最成熟的领域之一。理解睡眠结构是理解自动睡眠分期算法的前提。

**正常睡眠的"建筑结构"**：

一晚睡眠由 4-6 个**睡眠周期**组成，每个周期约 90 分钟：

```
Wake → N1 → N2 → N3 → N2 → REM → N2 → N3 → ... (循环)
```

| 阶段 | EEG 特征 | 持续时间 | 功能 |
|------|---------|---------|------|
| **N1**（入睡） | Alpha 消失，Theta 出现 | 5-10 min | 过渡 |
| **N2**（浅睡） | K-复合体 + 睡眠纺锤波（12-15 Hz burst） | 45-55% 总睡眠 | 记忆巩固开始 |
| **N3**（深睡/慢波睡眠） | 高振幅 Delta 波（<2 Hz, >75 µV） | 15-25% | 身体修复、免疫、记忆巩固关键阶段 |
| **REM**（快速眼动） | 低振幅快波（类似清醒）+ 快速眼动 | 20-25% | 做梦、情绪记忆处理、创造力 |

**为什么 EEG 是睡眠研究的黄金标准**：

只有 EEG 能区分这些睡眠阶段——fMRI 太吵不能在睡眠中用，且时间分辨率不够看到纺锤波和 K-复合体这些秒级事件。

**记忆巩固的"对话"模型**：

1. 白天学习 → 信息暂存在海马体
2. N3 深睡时 → 慢振荡（<1 Hz）组织"回放"窗口
3. 纺锤波（12-15 Hz）在慢振荡的上升相出现 → 打开皮层的"写入窗口"
4. 海马体的 sharp-wave ripples（100-250 Hz）在纺锤波内出现 → 具体记忆内容被传递到皮层

这个三层嵌套结构（慢振荡 > 纺锤波 > ripples）是 2024-2026 年睡眠记忆研究的核心范式。

### B3.3 注意力的神经机制——从鸡尾酒会效应说起

**为什么要学这个**：注意力调控是认知神经科学的核心课题，也是 BCI（被动 BCI / 神经人因学）和临床 EEG（ADHD 诊断）的基础。

**鸡尾酒会效应**：嘈杂的派对上，你能专注听一个人说话、忽略其他所有声音。但如果有人在远处喊你的名字，你立刻转头。这说明大脑同时在做两件事：
1. **自上而下的注意**：你主动选择关注一个声音（靠前额叶控制）
2. **自下而上的注意**：突出的刺激（你的名字）自动捕获注意（靠丘脑/顶叶）

**EEG 如何测量注意力**：

- **Alpha 抑制**：当你注意视觉左侧时，右枕叶的 alpha 功率下降（那个区域被"解锁"了），而左枕叶的 alpha 功率上升（那个区域被"静默"了）。Alpha 节律是大脑的"空闲信号"——活跃的脑区 alpha 低，空闲的脑区 alpha 高。
- **前额 Theta**：认知负荷增加（比如做心算）时，前额中线的 theta（4-8 Hz）功率增加。这是工作记忆资源占用的指标。
- **P300**：注意到一个罕见刺激时产生的 ERP 成分。P300 幅度反映注意资源的分配——任务越难、注意力越集中，P300 越大。

**实际应用**：
- **驾驶疲劳监测**：alpha 功率增加 + theta/alpha 比值变化 → 司机在犯困
- **ADHD 诊断**：theta/beta 比值在 ADHD 儿童中升高（有争议但仍在临床使用）
- **神经反馈训练**：实时反馈 alpha 或 theta 水平，训练用户调节自己的注意力状态

**关键直觉**：注意力不是"开/关"——它是一个**动态的资源分配过程**，在不同脑区之间不断调整。EEG 的高时间分辨率让我们能看到这个分配过程如何在毫秒尺度上变化。

## 为第五部分（EEG + 深度学习）的基础补充

### B4.1 为什么 EEG + DL 很难——和图像/文本的本质区别

**为什么要学这个**：很多 AI 研究者认为"EEG 也是信号，直接套 CNN/Transformer 就行了"。事实远非如此。

**EEG 数据的独特困难**：

| 困难 | 和图像/文本的对比 | 后果 |
|------|------------------|------|
| **极低信噪比** | 图像 SNR ~30-40 dB；EEG SNR ~0-10 dB | 需要大量平均/预处理才能提取信号 |
| **跨被试差异巨大** | 同一个 ImageNet 图片对所有人一样 | 你和我的 EEG 完全不同——头形、皮层折叠、电极阻抗都不一样 |
| **数据量极小** | ImageNet 1400 万张；NLP 万亿 token | 一个 EEG 实验通常只有 10-50 个被试、每人 1-2 小时 |
| **通道数/空间采样** | 图像百万像素 | EEG 最多 256 通道（且受体积导引限制） |
| **非平稳性** | 图片不随时间变化 | EEG 的统计特性随时间剧烈变化（打瞌睡了、注意力飘了） |
| **标注昂贵** | ImageNet 众包标注 | EEG 需要神经科学专家标注（或自报） |

**这意味着**：
- 直接把 ImageNet 上的 ResNet 迁移到 EEG **不工作**
- 需要专门为 EEG 设计的架构（如 EEGNet——故意做得很小、很浅）
- **预训练**（在大量无标签 EEG 上自监督学习）成为关键——这就是为什么 EEG Foundation Model 是 2024-2026 的热点

---

### B4.2 基础模型的核心思想——预训练为什么有效

**为什么要学这个**：LaBraM、NeuroLM、EEGPT 等 EEG 基础模型是当前最热的方向。理解"预训练"的逻辑才能理解这些模型在做什么。

**核心逻辑（以 NLP 为类比）**：

1. **阶段一：预训练（Pre-training）**
   - 用**大量无标签数据**训练模型完成一个"自我监督"任务
   - NLP 中：预测被遮住的词（BERT）或下一个词（GPT）
   - EEG 中：预测被遮住的脑电信号片段（masked neural code prediction）
   - 目标：让模型学会"什么样的 EEG 模式是合理的"——脑电的"语法"

2. **阶段二：微调（Fine-tuning）**
   - 用**少量有标签数据**训练模型完成具体任务
   - NLP 中：情感分类、问答
   - EEG 中：情绪识别、运动想象分类、癫痫检测
   - 因为预训练已经学到了 EEG 的通用表示，微调只需要少量数据就能适应新任务

**为什么 EEG 特别需要预训练**：
- 有标签的 EEG 数据极少（每个实验几十人，每人几百个 trial）
- 无标签的 EEG 数据相对多（临床 EEG 数据库如 TUH 有上万条记录）
- 预训练让模型先从无标签数据中学习"脑电的物理规律"，再用少量标注数据学习具体任务

**LaBraM 的创新**：它不是直接预测原始 EEG 波形（太嘈杂），而是先把 EEG 转成离散的"神经代码"（vector-quantized tokens），然后预测被遮住的代码。这类似于 BERT 预测被遮住的词——但"词"变成了"EEG 模式的量化表示"。

---

### B4.3 自监督学习入门——没有标签怎么学

**为什么要学这个**：EEG 基础模型的核心技术就是自监督学习。理解它的基本范式才能读懂相关论文。

**三大自监督范式**：

**1. 对比学习（Contrastive Learning）**

思路：让模型学会区分"相似"和"不相似"的数据对。

```
同一个人 + 同一状态 + 时间接近 = 正样本对（应该相似）
不同人 / 不同状态 / 时间遥远 = 负样本对（应该不同）
```

训练目标：在潜在空间中，正样本对的距离小，负样本对的距离大。

EEG 中的应用：BENDR 用这种方式学习"跨被试的不变表示"——不同人的同一种脑状态应该在潜在空间中靠近。

**2. 掩码预测（Masked Prediction）**

思路：遮住一部分输入，让模型预测被遮住的部分。

```
输入: [通道1: ████ | 通道2: OK | 通道3: OK | ... ]
目标: 预测通道1被遮住的区域
```

EEG 中的应用：LaBraM 遮住随机的 EEG 片段，让模型从周围的时间-通道信息推断被遮住的内容。

**3. 一致性学习（Consistency Learning）**

思路：同一段 EEG 的两种不同"视角"（比如不同的增强方式）应该产生一致的表示。

```
视角 A: 原始 EEG + 随机通道丢弃
视角 B: 原始 EEG + 随机时间偏移
目标: 两个视角的表示应该一致
```

**关键直觉**：自监督学习让模型从数据的**内在结构**中学习，不需要人告诉它"这是什么"。在 EEG 领域，这解决了"标注数据太少"的核心瓶颈。

---

### B4.4 脑解码的基本逻辑——从神经信号到语义

**为什么要学这个**：EEG-to-Text、DreamDiffusion 这些"科幻"般的工作是怎么做到的？理解基本逻辑，才能理解为什么它们目前还很初步。

**脑解码的核心假设**：

> 不同的心理内容（看到的图像、想到的词、感受到的情绪）对应不同的脑活动模式。如果我们能建立"脑活动模式 → 心理内容"的映射，就能从脑信号"读出"思维。

**三个关键步骤**：

1. **表示对齐（Representation Alignment）**：
   - 神经信号和语义内容在不同的"空间"里
   - 需要学习一个映射：EEG 表示空间 ↔ 语言/视觉表示空间
   - DeWave 把 EEG 编码成离散代码，然后对齐到 LLM 的词嵌入空间

2. **解码（Decoding）**：
   - 给定一段 EEG，通过对齐后的映射找到对应的语义内容
   - 可以是分类（从 N 个候选中选一个）或生成（自由生成文本/图像）

3. **验证（Validation）**：
   - 如何确认"解码"出来的内容是真的，而不是模型的幻觉？
   - ACL 2024 的关键论文 "Are EEG-to-Text Models Working?" 指出：很多看似成功的 EEG-to-Text 模型实际上在利用**统计捷径**（比如句子长度和 EEG 时长的相关性），并不是真的在"读脑"

**当前的现实水平**：
- **分类任务**（从 N 个图像中识别看到的那个）：60-80% 准确率
- **生成任务**（从 EEG 生成图像/文本）：能产生主题相关的模糊结果，但远未达到"读心术"
- **核心瓶颈**：EEG 的空间分辨率太低——你能看到"这个人在看人脸"，但看不到"这个人在看谁的脸"

**关键直觉**：脑解码不是"读心术"——它更像是从一个非常模糊的信号中提取统计规律。目前最诚实的描述是"我们能检测到粗粒度的认知状态变化"，而不是"我们能读出你在想什么"。

## 为第六部分（脑机接口）的基础补充

### B5.1 BCI 的基本闭环——人脑如何和机器对话

**为什么要学这个**：BCI 是 EEG 最引人注目的应用方向。理解基本闭环才能理解各种 BCI 范式的设计逻辑。

**BCI 闭环的四个组件**：

```
  用户大脑 ──(1)信号采集──> EEG 设备
       ↑                        ↓
  (4)反馈          (2)信号处理+特征提取
       ↑                        ↓
  显示器/机械臂 <──(3)分类+翻译──> 指令
```

1. **信号采集**：记录脑电（头皮 EEG 或皮层 ECoG）
2. **信号处理**：滤波 → 伪迹去除 → 特征提取（频段功率、CSP 空间模式等）
3. **分类/翻译**：把特征映射为指令（左/右/上/下 or 选字母）
4. **反馈**：把结果呈现给用户（光标移动、轮椅转向、字母出现）

**用户通过反馈学习调节自己的脑信号** → 机器通过数据学习识别用户的模式 → 双向适应（co-adaptation）。

**BCI 的"信息传输率"（ITR）**：

衡量 BCI 速度的关键指标：每分钟能传输多少 bit。

- **运动想象 BCI**：~20-30 bits/min（很慢——打字约 3-5 字/分钟）
- **P300 Speller**：~30-60 bits/min（15-30 字/分钟）
- **SSVEP**：~100-200+ bits/min（清华记录：60+ 字/分钟）

对比：正常打字速度约 200 字/分钟 ≈ ~600 bits/min。BCI 仍然比直接打字慢 3-30 倍。

---

### B5.2 运动想象的神经基础——想象动作时大脑发生了什么

**为什么要学这个**：运动想象（Motor Imagery, MI）是最成熟的 BCI 范式之一。理解神经基础才能理解为什么选择 mu/beta 频段作为特征。

**运动皮层的体感地图（Homunculus）**：

大脑中央沟两侧有一条"身体地图"——不同的身体部位由不同的皮层区域控制：
- 手：中央区中部偏侧方（C3 左手、C4 右手附近）
- 脚：中央区顶部（Cz 附近）
- 舌/面部：中央区侧下方

**"想象动作"激活的是同一张地图**。

当你实际移动右手时，左运动皮层（C3 附近）的 mu 节律（8-12 Hz）和 beta 节律（18-26 Hz）功率下降——这叫**事件相关去同步化（ERD）**。

当你只是**想象**移动右手（不实际动）——同样的 ERD 也会出现，只是幅度更小（约为实际运动的 30-50%）。

**BCI 就是检测这个 ERD**：
```
想象右手 → 左半球 (C3) mu/beta ERD
想象左手 → 右半球 (C4) mu/beta ERD
想象双脚 → 中线 (Cz) mu ERD
```

通过检测 ERD 出现在哪个区域，可以推断用户在"想"什么动作 → 翻译为指令。

---

### B5.3 稳态视觉诱发电位（SSVEP）——频率标记的巧妙利用

**为什么要学这个**：SSVEP-BCI 是目前最快的非侵入式 BCI，理解它的原理能体会 EEG 研究中频率编码的精妙。

**基本原理**：

当你盯着一个以固定频率闪烁的刺激（比如 12 Hz 闪烁的方块），你的视觉皮层会产生一个**和闪烁频率完全一致**的电信号——12 Hz 的正弦波出现在枕叶 EEG 中。不仅如此，还会出现谐波：24 Hz、36 Hz...

**这是大脑的"频率锁定"（frequency tagging）**——视觉皮层像一个共振器，被外部频率"驱动"。

**SSVEP-BCI 的设计**：

在屏幕上放 40 个方块，每个以不同频率闪烁（8.0, 8.2, 8.4, ..., 15.8 Hz）。

```
用户盯着目标方块（比如 10.6 Hz 那个）
    ↓
枕叶 EEG 出现 10.6 Hz 峰值
    ↓
算法检测频率峰值
    ↓
确定用户在看哪个方块 → 输出对应字符
```

**为什么 SSVEP 最快**：
- 不需要用户"学习"控制脑信号（MI-BCI 需要训练数周）
- 信号强且稳定（频率锁定是硬编码在视觉系统中的生理反应）
- 频率分辨率高：0.2 Hz 的间隔就够区分不同目标
- 清华大学团队用 FBCCA（滤波组典型相关分析）实现了 **60+ 字/分钟**

**关键直觉**：SSVEP 把大脑当成了一个"频率选择器"——你看哪个频率的闪烁，大脑就跟着那个频率共振。这种物理机制简单、信号强、可靠性高，因此 SSVEP-BCI 是目前最接近实用的非侵入式 BCI。

## 为第七部分（临床应用）的基础补充

### B6.1 癫痫的神经机制——为什么大脑会"短路"

**为什么要学这个**：癫痫检测/预测是 EEG 临床应用的核心场景，也是 AI for EEG 中数据最丰富的方向。

**正常 vs 癫痫脑电的区别**：

正常大脑中，兴奋和抑制保持动态平衡——就像城市的交通：有些路堵有些路通，但整体流动。

癫痫发作 = 这个平衡被打破。大量神经元**同时过度放电**，像交通系统突然所有路口同时绿灯——所有车同时冲出来，造成大面积瘫痪。

**在 EEG 上的表现**：
- **发作间期（Interictal）**：出现尖锐的"棘波"（spike）或"尖波"（sharp wave），持续 20-200 ms
- **发作期（Ictal）**：大面积高幅度节律性活动，通常从一个区域开始扩散到全脑
- **发作后（Postictal）**：全脑活动抑制，EEG 变平

**为什么 AI 在这里有效**：
- 模式明确（棘波有清晰的形态特征）
- 数据丰富（CHB-MIT 有连续多天的 EEG + 标注）
- 需求紧迫（人工读 EEG 需要训练有素的神经科医生，全球短缺）

**癫痫预测（不是检测）的难度**：
- 检测 = 发作已经开始了，从 EEG 中识别出来 → 已有 >95% 准确率
- 预测 = 发作还没开始，提前 30-60 分钟预警 → 仍然困难（假阳性率高）
- 预测的关键信号：发作前的微妙变化（同步性变化、频谱变化、高频振荡增加）

### B6.2 睡眠医学基础——正常睡眠的结构

**为什么要学这个**：自动睡眠分期是 EEG AI 最成熟的应用之一。这里补充临床视角。

**多导睡眠图（PSG）**：

睡眠研究的"黄金标准"检查不只是 EEG——它同时记录：
- **EEG**（至少 2 通道）：判断睡眠阶段
- **EOG**（眼电图）：检测快速眼动（区分 REM 和 N1）
- **EMG**（肌电图，通常下颌）：REM 时肌肉张力消失（肌肉麻痹）
- **呼吸气流、血氧、心电**：检测睡眠呼吸暂停等

**手工睡眠分期规则（AASM 2017）**：

每 30 秒（一个 epoch）标记为一个睡眠阶段：
- **N1**：>50% alpha 消失，出现 theta
- **N2**：出现 K-复合体或睡眠纺锤波
- **N3**：>20% 的 epoch 是慢波（<2 Hz, >75 µV）
- **REM**：低振幅混合频率 + 快速眼动 + 肌肉张力消失

一个训练有素的技师标记一整晚 PSG（~8 小时 = ~960 个 epoch）需要 2-4 小时。**这就是自动睡眠分期的直接动机**——用 AI 做到同样的事，只需要几秒。

**关键临床指标**：
- 睡眠效率 = 睡眠时间 / 在床时间（正常 >85%）
- 各阶段占比（N3 偏低 → 深睡不足 → 白天困倦）
- 睡眠潜伏期（入睡时间，正常 <30 min）
- REM 潜伏期（入睡到第一次 REM，正常 ~90 min；抑郁症患者缩短）

### B6.3 脑年龄的概念——为什么大脑会"老"得不一样快

**为什么要学这个**：脑年龄是一个简洁但强大的生物标志物概念，且完全可以用 EEG 测量。

**核心概念**：

训练一个模型，输入：一个人的 EEG；输出：这个人的年龄。在健康人群上训练后，这个模型学到了"不同年龄的 EEG 应该长什么样"。

然后用它预测新的人：
- **预测年龄 ≈ 实际年龄**：这个人的大脑老化速度正常
- **预测年龄 > 实际年龄**（脑年龄差 > 0）：这个人的大脑"比实际年龄老"——可能是疾病早期信号
- **预测年龄 < 实际年龄**（脑年龄差 < 0）：这个人的大脑"比实际年龄年轻"

**EEG 中和年龄相关的变化**：
- Alpha 峰频率随年龄降低（青年 ~10 Hz → 老年 ~8 Hz）
- Delta/theta 功率随年龄先降后升
- 脑网络的复杂度指标随年龄变化
- 脑连接模式改变

**为什么脑年龄差有用**：
- 脑年龄差 > 0 与**死亡风险增加**相关（Engemann et al., 2022）
- 脑年龄差 > 0 在**认知衰退**出现前就能检测到
- 可以用于药物试验的终点指标——药物是否减缓了脑年龄增长？

**关键直觉**：脑年龄把一个高维的、复杂的 EEG 信号压缩成一个数字——"你的大脑看起来像几岁"。这个数字虽然粗糙，但因为是在大量人群数据上学到的，所以包含了关于脑健康状态的重要信息。

## 为第十部分（新兴方向）的基础补充

### B9.1 多模态融合的物理基础——EEG+fMRI 为什么互补

**为什么要学这个**：多模态脑成像（同时记录 EEG 和 fMRI）是一个技术挑战极高但科学回报也极高的方向。理解"互补"的物理原因。

**EEG 和 fMRI 测量的是完全不同的物理量**：

| | EEG | fMRI |
|---|---|---|
| **测什么** | 神经元的电活动（突触后电位产生的电场） | 血液动力学反应（局部血流、血氧变化） |
| **时间分辨率** | ~1 ms | ~1-2 秒 |
| **空间分辨率** | ~1-2 cm（头皮）/ ~5 mm（源定位） | ~1-3 mm |
| **反映的时间尺度** | 即时的神经活动 | 延迟 4-6 秒的代谢反应 |

**互补性**：
- EEG 告诉你**什么时候**发生了什么（精确到毫秒）
- fMRI 告诉你**在哪里**发生了什么（精确到毫米）
- 两者结合 = 既知道在哪里，又知道什么时候

**技术挑战**：在 MRI 扫描仪里记录 EEG 极其困难——强磁场（3T）会在 EEG 电极上感应出巨大的伪迹（振幅是真实 EEG 的 100-1000 倍）。需要专门的 MR-compatible EEG 设备和复杂的伪迹去除算法。

**2024-2026 前沿**：用 LLM（如 Gemma、Llama）联合处理 EEG 和 fMRI 的嵌入表示——这是多模态 AI 的新方向。

### B9.2 生成模型用于脑科学——为什么要"生成"EEG

**为什么要学这个**：2024-2026 年，扩散模型（Diffusion Model）开始被用于 EEG 领域。为什么要"生成"脑电？

**不是为了"造假"——而是为了解决实际问题**：

**1. 数据增强（Data Augmentation）**

EEG 数据量太小。传统增强方法（加噪声、时间拉伸）效果有限因为 EEG 的时频结构复杂。
生成模型可以产生"看起来像真实 EEG、保留时频特征、但不是任何真实记录的副本"的合成数据。用合成数据训练 → 模型泛化性提升。

**2. 隐私保护**

临床 EEG 包含敏感信息（可能识别个人身份，可能暴露健康状况）。共享原始 EEG 有伦理和法律障碍。
生成模型可以产生"统计上等价但不对应任何真实个体"的合成 EEG → 可以安全共享。

**3. EEG 去噪**

用扩散模型将"含噪 EEG"逐步"去噪"到干净 EEG（类似于图像去噪的 DDPM 方法）。
EEGDfus (2025) 展示了这种方法可以替代传统的 ICA 去噪。

**4. 缺失数据补全**

如果一个电极坏了或者记录中断，生成模型可以从其他通道的信息"想象"出缺失通道应该有什么信号。

**关键直觉**：生成模型在 EEG 中的角色不是"创造新的大脑信号"，而是作为一种**统计工具**——建模 EEG 数据的分布，然后利用这个分布来做增强、去噪、补全、和隐私保护。
